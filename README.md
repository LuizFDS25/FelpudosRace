# Felpudo's Race
Felpudo's Race é um jogo onde você deve desviar dos inimigos tanto no chão quanto no céu pulando e agaixando
seu objetivo é sobreviver a essa corrida mortal por 1:00 minuto!!!
Mesmo com mecânicas simples Felpudo's Race é um jogo muito desfiador, será que você consegue vencer?

//Recursos//
O jogo possui um cenário com efeito parallax para maior imersão
timer reativo que aumenta a dificuldade com o passar do tempo tornando a experiancia mais desafiadora 
mudanças de hitbox em tempo real com base nas animações

//Controles//
Tecla "S" - Abaixar
Tecla de espaço - Pular

//Como Jogar//
O objetivo do jogo é esquivar dos inimigos que vão aparecendo no cenário. Pule pelos inimigos do chão e abaixe-se para se esquivar dos inimigos voadores.
O jogador deve sobreviver por 1 minuto completo para ganhar o jogo, sendo que a frequencia e a velocidade dos inimigos aumenta a cada segundo até o final.
Ao começar a partida o jogador possui nove vidas, ao chegar a zero é game over, assim, precisando recomeçar o timer do zero.
